Project Overview

Cheap Grocery is a simple web-based project that displays grocery items at affordable prices.
It is built using basic HTML, CSS, and JavaScript and demonstrates a beginner-level e-commerce interface.

🚀 Features
🏠 Simple and clean homepage
🛍️ Display of grocery products
🛒 Add to Cart functionality (basic counter)
📱 Responsive design (basic)
🎨 Styled using CSS
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
📂 Project Structure
Cheap-Grocery/
│
├── index.html    # Main webpage
├── script.js     # JavaScript functionality
└── README.md     # Project documentation

Download or clone this repository
Open the folder
